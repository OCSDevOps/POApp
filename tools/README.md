# Tools

This folder keeps non-production scripts organized by purpose.

## Supported Folders

- `diagnostics/` one-off inspection helpers for routes, schema, and data verification
- `maintenance/` local helpers such as cache clearing and storage-link setup
- `dev/` local server helpers and development launch scripts

## Reference-Only Material

- `archive/legacy-db/` historical database repair and seeding scripts
- `archive/sqlserver-reference/` old SQL Server reference scripts

Anything under `tools/archive/` is not part of the supported Laravel migration and seed workflow.
